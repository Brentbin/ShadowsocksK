# Shadowsocks K

![Icon](./mobile/src/main/res/mipmap-xhdpi/ic_launcher.png)

Shadowsocks client for Android using Kotlin.

## Feature
* only support android api 21+ (5.x+)
* only support OnePlus 3T device which android api is 25 (7.1.x)
